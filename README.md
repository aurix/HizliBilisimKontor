# Bu program sayesinde ne yapabilirsiniz?

- Yazılan vergi numarasına göre, Firmaya kolayca kontör yüklemelerinizi yapabilirsiniz.

# Program nasıl kullanıma uygun hale getirilir?

- Programın içerisinde ki username, password, apikey, secret alanlarını Hızlı Bilişim'e girerken kullandığınız Kullanıcı adı ve Şifre'ye göre doldurunuz. Apikey ve SecretKey'leri Hızlı Bilişimden temin ediniz. Doldurup programı derledikten sonra, programı kullanmaya başlayabilirsiniz.
